# Batform - Social Media Website

A clean and professional social media platform built with HTML, CSS, and ReactJS. More functionalities will be available very soon

## Project Overview

Batform is a social media website that demonstrates modern web development practices using semantic HTML5, clean CSS design, and React for interactivity. The project follows a progressive enhancement approach: HTML for structure, CSS for presentation, and React for dynamic functionality.
